# Production Dashboard

This folder contains all widgets and pages specific to the Production dashboard UI.

## Recent Updates (May 2025)

- **Provider Registration**: `ProductionJobProvider` has been registered in `main.dart` to manage production job states and workflows.
- **Service Integration**: `ProductionService` is configured to handle backend communication for production-related operations.
- **Features**:
  - Job list management
  - Labor assignment
  - Job status updates
  - Production tracking
