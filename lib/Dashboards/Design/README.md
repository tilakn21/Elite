# Design Dashboard

This folder contains all widgets and pages specific to the Design dashboard UI.

## Recent Updates (May 2025)

- **State Management**: `JobProvider`, `ChatProvider`, and `UserProvider` are registered in `main.dart` to manage the state for design-related operations.
- **Service Layer**: `DesignService` is integrated to handle backend communication for design jobs and chat functionality.
- **Backend Readiness**: The module is structured to support future API integration for design job management and real-time chat features.
