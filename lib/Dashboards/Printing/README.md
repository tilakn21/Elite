# Printing Dashboard

This folder contains all widgets and pages specific to the Printing dashboard UI.

## Recent Updates (May 2025)

- **Provider Registration**: `PrintingJobProvider` has been registered in `main.dart` to manage printing job states and operations.
- **Service Integration**: `PrintingService` is configured to handle backend communication for printing-related tasks.
- **Features**:
  - Job assignment to labor
  - Quality check workflows
  - Real-time status updates for printing jobs
